🪨 Rock, Paper, Scissors - Python Edition
A lightweight, interactive command-line game built with Python. This project showcases clean logic, input sanitization, and the use of Python's built-in random module to simulate a computer opponent.

🚀 Features
Case-Insensitive Input: Users can type "ROCK", "rock", or "RoCk" without the game breaking.

Input Validation: The program checks if the user's choice is valid before playing.

Randomized AI: Uses the random library to ensure the computer’s choice is unpredictable.

Instant Feedback: Clearly displays the computer's choice and the final result of the round.

🛠️ Tech Stack
Language: Python 3.x

Modules: random (Standard Library)

📦 Installation & Usage
Clone the repository:

Bash
git clone https://github.com/ashishiy31/rock-paper-scissors-python.git
Navigate to the directory:

Bash
cd rock-paper-scissors-python
Run the game:

Bash
python game.py
🧠 Logic Breakdown
The game follows the standard rules:

Rock beats Scissor

Paper beats Rock

Scissor beats Paper

The program uses a conditional if-elif-else structure to compare the user_choice against the comp_choice.
